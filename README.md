# Science Week Web System (Next.js Version)

This is the web system for "Science Week, Wachirawit School". It has been migrated to use a modern stack with [Next.js](https://nextjs.org), Prisma, and Tailwind CSS.

## 📖 Documentation
Please refer to the following files for in-depth system documentation:
- [System Documentation](./system_documentation.md) - Contains the original and detailed architecture of the system (including Google Apps Script API, Database structure on Google Sheets, etc.)
- [AppScript References](./science-week-github-pages-appscript.md) - For the AppScript backend.

## 🚀 Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠 Technology Stack
- **Framework:** Next.js (React)
- **Database ORM:** Prisma
- **Styling:** Tailwind CSS, Material UI (MUI), Emotion
- **Components:** SweetAlert2, React Calendar, React Date Picker
